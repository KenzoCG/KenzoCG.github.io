Editors

Visual Studio Code (VS Code) is one of the easiest and most popular choices for Python development today. It offers many free extensions that enhance Python support within the editor. Another option (but more as a side companion) is Jupyter Notebooks. Jupyter Notebooks is a web-based interface for Python that allows running code in isolated blocks rather than executing an entire script. All said though, you will most likely want to use VS Code for writing your software and maybe Jupyter Notebooks for testing blocks of code.

For VS Code you will also want to make sure you install the python extensions. If you have not installed python yet, I would recommend doing that first so the extensions can auto-detect python for you.

Python extensions for VS Code

You can also press Ctrl + Shift + X while in VS Code to open the extensions tab and then search for the extension ID.

ms-python.python

The extension will also install two other extensions for you automatically.

  • Pylance - to provide performant Python language support
  • Python Debugger - to provide a seamless debug experience with debugpy

Python Distributions

Some applications embed a Python interpreter within their source code (such as Blender), meaning you may not need to install Python separately on your machine. However, it's still a good idea to install Python separately, as your code editor will likely require it for IntelliSense and debugging.

  • Python Downloads Page

Follow the guides below for your operating system if you need further assistance. Note the important thing to do is, make sure when you run the python installer that you check the box to enable system paths. Enabeling system paths will let your operating system find python when you type the command python into the terminal.

After installing the python interpreter, you can check if you operating system can find the path. Open the terminal and type python, then press enter to see if the interpreter starts. Also be sure to close any terminals you had open during installation as they will not be updated.
